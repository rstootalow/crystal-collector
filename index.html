Crystal Collector Game

    Instructions

  • You will be start the game with a random number at the start of the game
  • There are four crystals below. Each crystal will be assigned a random value at the start of each new game
  • When you click on the crystal, the number of points for that crystal will be assigned to your total score
  • Click on the crystals to try and match the number of points provided at the start of the game
  • To reiterate, the number of points for each crystal will change at the start of each new game
  • Good Luck!
